# JERA TEST PROJECT - STUDY LINKS #

## WEB API ##

### Prerequisites ###

* JDK 8
* MAVEN 3.X 

### Running the App

```
mvn spring-boot:run
```


## UI - USER INTERFACE ##

### Prerequisites ###

* NPM
* Angular CLI

### Running the App

Install node modules
```
npm install
```

Run

```
ng serve
```

Check the application on: http://localhost:4200/
